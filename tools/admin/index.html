🎙️ 音檔轉逐字稿
Groq Whisper Large v3・中文優化・自動壓縮大檔
① 一次性安裝
pip install groq
② 轉錄模式
載入中...
③ 貼上資料夾路徑（批次轉裡面所有音檔）
④ 目的地資料夾（轉完後搬移音檔＋逐字稿）
⑤ 下載並雙擊執行
python transcribe.py "C:\OBS" --dest "C:\OBS\轉檔完成"
⑥ 轉錄完成後，逐字稿在目的地資料夾，貼給 Claude
用 transcript-training-pack 處理 C:\OBS\轉檔完成\檔名_逐字稿.md
工具位置：C:\Users\admin\Desktop\classroom\transcribe-audio\
📝 transcript-training-pack
單一逐字稿 → 培訓報告＋摘要＋心智圖
① 輸入來源
貼上逐字稿全文、單一檔案，或含逐字稿的資料夾路徑。
優先找檔名含：逐字稿、transcript、Transcript
② 命名候選
先根據內容提出 2-4 個候選課程名或檔名，讓使用者選，再正式命名。
③ 清理逐字稿
讀 frontmatter、標題、來源、日期。
正文取 ## Transcript 後的內容。
清口語贅詞、重複句，不改原意。
保留：段落順序、時間軸、概念框架、操作流程、比較結構。
④ 產出 4 份成品
培訓報告.md（6 章）：學習目標、關鍵概念、參與者理解度、練習與活動、資源與材料、後續培訓

摘要.md（4 章）：摘要總覽、待辦項目、重點記錄、建議行動

心智圖.md — 保留足夠密度，不是只剩骨架

心智圖網頁版.html — Markmap HTML
⑤ 輸出位置
C:\Users\admin\Desktop\本機MD檔\30_研究\課程逐字稿整理\YYYYMMDD_課程名\
⑥ 品質檢查
確認四個檔案都有產生、章節完整、比較框架和操作細節有保留。
不能亂補外部資訊。
用途：把文章/影片逐字稿轉成摘要成品，適合學習筆記整理。
📚 course-filing
課程歸檔 Obsidian，含知識點拆解＋日記更新
① 收集資訊
確認：課程名稱、領域分類、筆記/逐字稿/心智圖三份資料。
② 決定路徑與命名
整理到：
30_研究/[領域]/YYYYMMDD課程名稱/
③ 整理主筆記
做成結構化筆記，補 frontmatter、警告 callout、心智圖 wikilink。
④ 處理逐字稿
一份完整逐字稿存到 30_研究。
另外從逐字稿抽概念，建立 40_知識庫 的原子知識筆記。
⑤ 搬移心智圖
把心智圖放進同一課程資料夾。
⑥ 更新日記
在當日日記記錄這次歸檔與新增知識點。
用途：連摘要、逐字稿、心智圖、知識點、日記一起歸檔到 Obsidian 長期知識庫。
