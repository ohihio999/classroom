🎙️ 音檔轉逐字稿
Groq Whisper Large v3・中文優化・自動壓縮大檔
① 一次性安裝
pip install groq
② Groq API Key（只需填一次，不會上傳）
③ 貼上資料夾路徑（批次轉裡面所有音檔）
④ 下載並雙擊執行
python transcribe.py "C:\OBS"
⑤ 轉錄完成後，逐字稿在同一資料夾，貼給 Claude
用 transcript-training-pack 處理 C:\OBS\檔名_逐字稿.md
